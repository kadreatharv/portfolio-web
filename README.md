# 🌌 Atharv Kadre | Creative Developer & Cyber Security Enthusiast

Welcome to my premium portfolio website. This project is a high-fidelity, interactive 3D experience built to showcase my expertise in **Web Development**, **Cyber Security**, and **DevOps**. 

Built with a focus on cinematic aesthetics and smooth performance, this portfolio pushes the boundaries of modern web design.

---

## ✨ Features

- **🛡️ Cyber-Themed Design**: A sleek, dark, and futuristic "Blood Moon" aesthetic with custom glowing elements and neon accents.
- **🎮 Interactive 3D Avatar**: A high-fidelity 3D model that tracks the cursor and blinks, creating a lifelike presence.
- **🧊 Physics-Driven Tech Stack**: An interactive "Tech Stack" page where technologies are represented as 3D blocks with real physics using `react-three-rapier`.
- **🚀 Ultra-Smooth Navigation**: Seamless scrolling and page transitions powered by **Lenis** and **GSAP**.
- **📱 Responsive Layout**: Fully optimized for a premium experience across all modern browsers and devices.
- **🎨 Cinematic Typography**: Bold, high-contrast typography using custom fonts like *Clash Display* and *Satoshi*.

## 🛠️ Tech Stack

### Frontend
- **Framework**: [React](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: Vanilla CSS with CSS Variables for a robust design system.

### 3D & Animations
- **3D Engine**: [Three.js](https://threejs.org/)
- **React Bridge**: [@react-three/fiber](https://github.com/pmndrs/react-three-fiber)
- **Helpers**: [@react-three/drei](https://github.com/pmndrs/drei)
- **Physics**: [@react-three/rapier](https://github.com/pmndrs/react-three-rapier)
- **Animations**: [GSAP (GreenSock)](https://greensock.com/gsap/)
- **Smooth Scroll**: [Lenis](https://github.com/studio-freight/lenis)

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18+)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/kadreatharv/portfolio-web.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd portfolio-web
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Build for production:**
   ```bash
   npm run build
   ```
